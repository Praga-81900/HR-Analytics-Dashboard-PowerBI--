# HR Analytics Dashboard 

## Project Overview
This project analyzes employee attrition data to identify key factors contributing to employee turnover. Built using *Microsoft Power BI*, it helps HR teams make data-driven decisions.

##  Tech Stack
- *Tool:* Microsoft Power BI And SQL(Structured Query Language)
- *Techniques:* Data Cleaning, DAX Measures, Data Modeling, Data Visualization.

##  Key Insights
- *Total Employees:* 1,470
- *Attrition Rate:* 16.1% (High concern)
- *Key Factor:* Employees in "Sales" and "Lab Technician" roles have higher attrition.
- *Age Group:* Young professionals (19-25) are leaving more frequently.
